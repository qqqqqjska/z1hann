把你想用于 520 信封/信纸的字体文件放到这个文件夹。

当前会按下面顺序自动尝试读取：
1. assets/fonts/520-letter.woff2
2. assets/fonts/520-letter.otf
3. assets/fonts/520-letter.ttf

你可以直接：
1. 把你的字体文件重命名成上面其中一个名字后放进来
2. 推荐优先用 `520-letter.otf` 或 `520-letter.ttf`
3. 或者告诉我你的字体文件名，我帮你把代码路径改掉

支持的常见字体格式：
- .ttf
- .otf（如果你用 otf，告诉我，我顺手帮你把代码里的 format 一起改掉）
- .woff2
